Vue Dialog弹窗解决方案

在做业务代码的modal弹窗时，总是围绕visible变量以及控制visible变量逻辑，能否简化弹窗相关逻辑呢？

该方案配合ElementUI或AntdV等组件库的modal组件更佳。
